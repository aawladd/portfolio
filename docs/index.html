Research statement

Framing the work with academic seriousness

My work explores how computational intelligence can become more rigorous, interpretable, and useful for engineering and scientific systems.

I am particularly interested in combining numerical simulation, machine learning, deep learning, scientific machine learning, robotics, and control theory to create methods that respect physical structure while remaining practically effective. This direction supports both methodological innovation and real-world deployment.
