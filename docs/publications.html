Publication strategy

Presenting outputs with clarity and consistency

A strong publication page should make it easy to scan titles, venues, years, status labels, and contribution summaries. It should also connect each output to supporting materials when possible, including PDF files, code repositories, data links, or presentation slides.

Recommended content
  • Journal articles, conference papers, and workshop papers
  • Preprints, theses, technical reports, and book chapters
  • Code, data, slides, posters, and supplementary materials
  • One-line contribution notes for each output
