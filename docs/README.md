# Research-Centered Academic Website

A darker, more premium, more academic static website for **Md Awlad Hossain** or any researcher who wants a GitHub Pages–ready site focused on:

- research agenda
- publications
- projects and prototypes
- teaching and mentorship
- research notes, YouTube, and blogs
- academic profile links and CV

## Tech stack

- HTML
- CSS
- JavaScript
- jQuery via:
  `https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js`

## Pages

- `index.html` — research-centered homepage
- `research.html` — research agenda and methods
- `publications.html` — papers, manuscripts, scholarly outputs
- `projects.html` — research systems and technical projects
- `teaching.html` — teaching philosophy and mentorship
- `youtube.html` — research notes, videos, blogs, and technical communication
- `contact.html` — academic links and CV
- `404.html` — fallback page

## Main file to edit

Start with:

- `assets/js/site-data.js`

That file controls most of the editable content, including:

- name
- title
- subtitle
- research themes
- projects
- publications
- teaching areas
- videos
- blog notes
- contact links
- CV path

## CV file

Replace:

- `assets/files/replace-with-your-cv.pdf`

with your real CV file, then update the filename in `assets/js/site-data.js`.

## GitHub Pages deployment

1. Create a GitHub repository.
2. Upload all files from this project.
3. Go to **Settings > Pages**.
4. Set the source branch to `main` and the folder to `/root`.
5. Save and wait for GitHub Pages to publish.

## Suggested next edits

- add your real Google Scholar, ORCID, GitHub, LinkedIn, and YouTube links
- replace placeholder publication items with real papers

- add a professional profile image later if you want a more personal hero section

## Notes

This version is intentionally more **academic and research centric** than a generic portfolio. The homepage and inner pages are written to feel closer to a serious researcher website or lab-style profile rather than a startup or personal brand template.
